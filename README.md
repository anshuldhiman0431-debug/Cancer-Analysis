#  Cancer Data Analysis

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a cancer dataset to identify patterns and trends in patient demographics, cancer types, stages, and treatment costs.

The dataset spans from **2015 to 2024**, and the goal is to extract meaningful insights using data analysis and visualization techniques.

---

## Objectives
- Understand the distribution of cancer cases across age, gender, and countries  
- Analyze different cancer types and stages  
- Study treatment cost patterns  
- Generate insights using data visualization  

---

##  Dataset Information
The dataset includes the following features:

- Age  
- Gender  
- Country  
- Cancer Type  
- Cancer Stage  
- Treatment Cost  

---

##  Technologies Used
- **Python**  
- **Pandas** (Data manipulation)  
- **NumPy** (Numerical operations)  
- **Matplotlib** (Data visualization)  
- **Seaborn** (Statistical visualization)  
- **Jupyter Notebook**  

---

##  Project Workflow

### 1. Data Loading
- Imported dataset using Pandas  
- Checked dataset structure and column information  

### 2. Data Cleaning
- Handled missing values  
- Removed duplicate entries  
- Ensured correct data types  

### 3. Exploratory Data Analysis (EDA)
- Generated summary statistics  
- Analyzed numerical and categorical features  
- Identified trends and patterns  

### 4. Data Visualization
- **Histogram & KDE plots** → Age distribution  
- **Bar charts** → Cancer types & stages  
- **Pie charts** → Gender distribution  
- **Country-wise analysis**  
- **Treatment cost distribution**  

---

##  Key Insights
- Patient age ranges from **20 to 89 years**, with an average around **54 years**  
- Cancer cases are distributed across multiple types and stages  
- Gender distribution is relatively balanced  
- Significant variation exists in treatment costs  
- Cancer occurrence varies across countries
